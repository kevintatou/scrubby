Scrubby
Local First, Offline
Clipboard Safety

Scrub secrets before they hit your AI prompts.

Scrubby is a local-first CLI that sanitizes your clipboard and replaces sensitive data with clear placeholders. No cloud. No telemetry. Explicit run only.

Install and Run
Example
Input: Email a@b.com IP 10.0.0.1 UUID 123e4567-e89b-42d3-a456-556642440000 JWT aaaa.bbbb.cccc TOKEN AbCDeF0123456789AbCDeF0123456789 Output: Email <EMAIL> IP <IP> UUID <UUID> JWT <JWT> TOKEN <TOKEN> Summary: Scrubby cleaned your clipboard: - Emails: 1 - IPs: 1 - UUIDs: 1 - JWTs: 1 - Tokens: 1 Safe to paste.

Clipboard First

One command or one hotkey. It scrubs what you already copied, then puts it back safely.

Private by Design

Scrubby never touches the network. All detection happens locally.

Focused Scope

Emails, IPs, UUIDs, JWTs, and high-entropy tokens. Simple placeholders that read cleanly.

Install

Local Build
cargo build --release ./target/release/scrubby --clipboard
Install Script
curl -fsSL https://example.com/scrubby/install.sh | sh Coming soon.

Modes

Shortcut Mode
scrubby --clipboard

Bind it to a custom hotkey for instant scrubbing.

Watch Mode (Experimental)
scrubby --watch scrubby --watch --interval-ms 500

Polling watch mode for users who want automatic scrubbing.

Pro (Planned)

Pro is planned, not shipped yet. The roadmap includes stable placeholders, JSON reports, config files, and file/stdin mode.
